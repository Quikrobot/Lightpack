Установка плагина
Скачать плагин - http://lightpack.googlecode.com/hg/XBMC/script.lightpack/script.lightpack.zip
Для установки из XBMC (install plugin from .zip) просто укажите путь к скачанному архиву. 

Плагин работает с XBMC 11 Eden, будет и в XBMC 10 Dharma, но с урезанным функционалом.

Обновления будут происходить автоматически
