Это исходные коды плагина визуализации для проигрывателя Winamp. 
Автор: Евгений Беляев
Демонстрация работы: http://vimeo.com/28810239

Для компиляции исходников необходима библиотека GLScene. Скачать ее можно тут: http://glscene.ru/download.php?view.542  снимок от 04.05.2010.
Плагин я писал в RAD Studio 2010, но думаю с более ранними версиями Дельфи тоже скопилируется нормально.
Также необходима библиотека RXLib http://rx.delphiplus.org/downloads/lib/rxlib-2.7.7.5d7-2010_psa1974.zip
