Плагин визуализации для AIMP
Автор: Николай Всильчук

Плагин просто отображает уровень по каналам (1-5 светодиоды для левого канала, 6-10 для правого).

LightPack.pas - SDK для Delphi. Команды максимально приближены к питоновским.
LightPack_SDK_Demo - Простенькая демка работы SDK.
aimp_LightPack - Плагин для AIMP2.
