PocketBook 902 application using Lightpack API over WiFi

Download SDK from http://pocketbook-free.sourceforge.net/

Compile application and upload to device:

$ cp -r "PocketBook LightpackDrivePB" "SDK/sources/PocketBook LightpackDrivePB"
$ cd "SDK/sources/PocketBook LightpackDrivePB"
$ ./makearm.sh
$ cp LightpackDrivePB /pocketbook902/applications/LightpackDrivePB.app
