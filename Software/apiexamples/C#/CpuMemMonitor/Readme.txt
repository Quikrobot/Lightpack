CpuMemMonitor - файлы проекта на C#
CpuMemMonitor.exe - исполняемый фаил
Run.bat - исполняемый файл запускать через него.

Программа отслеживает загрузку ЦП и используемую память и сопровождает это визуализацией(для 10 диодов, расположенных по часовой стрелке с началом в левом нижнем углу).

Обработана ошибка, возникающая при отсутствующем Лайтпаке.

Добавлена возможность изменения gamma и smootch, задаются в Run.bat.
