Исходники программы, которая представляет собой индикатор громкости звука с визуализацией через Лайтпак.
Автор: Владислав Кочемаев
Репозиторий автора: http://code.google.com/p/vladdev/source/browse/#svn%2Ftrunk%2FVolumeLight
Вики: http://code.google.com/p/vladdev/wiki/VolumeLight

Используется библиотека NAudio, которая предоставляет удобный .NET-интерфейс для нативного Core Audio API, а также libLightpack для .NET. Программа будет правильно работать под Windows 7, под другие версии ОС нет гарантий, также необходим .NET Framework 3.5.
